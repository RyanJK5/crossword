How to use
- Open `index.html` directly in your browser (double-click the file).

Start a simple static server (recommended during development):

PowerShell / Windows (Python 3 must be installed):

```powershell
cd c:/Users/Ryan/Documents/repositories/other/crossword
python -m http.server 8000
```

Then open `http://localhost:8000/` in your browser.
